New Home Page

The current home page is temporary while the site is under construction. The future home page should contain:

  • Control to select language.

    The user can select to see the site in their own language.

  • Brief project overview

    A paragraph or so and links to more information.

  • Register and Log-in links

    These are in the top right of all pages. After logging in, the user's name is displayed here.

  • Search area for participating entities

    Here users can see if their own local government entity is registered. They can also browse participating entities.

  • Area to highlight new and most active.

    This includes both volunteers and government entities. It will also show statistics on site use, by region, etc.

  • Call to action

    Links urging users to register themselves and their local government entity.
