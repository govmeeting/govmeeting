Further Goals

  • Make this software usable by the estimated 450,000 cities and towns worldwide.
  • Grow a database of the real issues as defined by communities themselves.
  • Educate and involve more people in the workings of democracy.
  • Serve as a foundation on which to build other more active features.
  • Learn lessons at the local level that can translate to state and federal levels.
