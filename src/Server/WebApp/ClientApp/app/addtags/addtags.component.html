Add topic & section tags


Step 3 is to assign "tags" to parts of the meeting:
  • SECTION TAGS: Read Minutes, Introduce Bills, Public Comment, etc.
  • TOPIC TAGS: "Pave Main St.", "Hire policemen", "Parking fees", etc.
To add a topic tag:
  • Highlight a text phrase to use it as the topic.
  • Or select a topic name from recent past toics.
  • Click up/down arrows to move tag.
Section tags:
  • Pre-processing attempts to assign these.
  • Click up/down arrows to move if incorect.
