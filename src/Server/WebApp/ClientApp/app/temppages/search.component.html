Search Meetings

After the meeting information is in a relational database, it can be used in many ways.

Search for information

  • Search all past meetings for discussion on a specific issues.
  • Search for everthing that a specific official said on an issue.
  • Search for when specific bills were either discussed or acted upon.
  • Search for anything on a specific type of issue (public safety, zoning, etc)

Browse information

  • Browse all topics discussed in the past year.
  • See which officials attended each of the past meetings.
  • See what legislation was voted on and what the votes were.
  • Browse all public commentary on specific issues.

Set alerts

You can sign up for alerts, via email or text messages. For example an alert can be sent:

  • when any specific issue is discussed at a meeting.
  • when a there is discussion in any category of issues (Recreation, Zoning, Animal Control, etc).
  • when any new legislature is proposed.
  • whenever any search criteria that you set up returns results in the future.

Comparisons with other communities

  • Compare types of issues that are discussed.
  • Compare how similar issues are handled.
  • Compare meeting structures and time allotment.
  • Compare procedures for public comment.
