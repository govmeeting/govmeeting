Govmeeting - Software to increase involvement in government meetings

This will eventually become the home page for the software being developed at: github.com/govmeeting/govmeeting
Eventually the home page will contain (among other things):

  • A search box and directory of towns/cities who are participating
  • A listing of the meetings in the database.
  • An area highlighting new particpants
  • An area highlighting active community volunteers


The Meeting tab above shows the start of a program for displaying a meeting transcript.
The transcript is from a selectmen's meeting in a small town in Maine.
A user can click on a topic to only see only what was said on that topic. Or they can click on a name to see only what that person said.
Right now we don't yet show the context surroundng what the selected person said.

The Add Tags tab above shows the start of a program that a community volunteer will use to process a transcript.
To make transcripts more useful, meaningful topic and section names need to be assigned to the parts of the meeting.
