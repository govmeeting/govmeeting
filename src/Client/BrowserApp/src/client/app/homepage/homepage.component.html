Govmeeting - Software for government involvement

This will eventually become the home page for the software being developed at: github.com/govmeeting/govmeeting
The tabs above show the start of some of the features being developed.

The Fix Errors tab shows the start of a program for correcting errors in a transcript. The transcript was created automatically from a video recording.

The Add Tags tab shows the start of a program for adding topic and section names to a transcript.
This sample transcript is of a Philadelphia City Council meeting.

The Meeting tab shows the start of a program for displaying a meeting transcript.
The transcript is from a selectmen's meeting in a small town in Maine.
