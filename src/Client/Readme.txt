This is is the client side piece of the public facing website, where anyone can access
the information in the Govmeeting database.

PROJECTS

BrowserApp - Single Page Application written in Typescipt with AngularJS.
	This app is currently using a static JSON file to mock the server WebApi component which would supply this data. 

		If you are developing on Linux or Mac, you can ignore files that are specific for using Visual Studio on Windows:
		packages.config - mainly NuGet packages used by Visual Studio for intellisense.
		web.config - server setting if using a Microsoft server. 


BrowserApp_Tests - unit tests of BroswerApp code using Jasmine.
